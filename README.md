# React Counter App

This is a simple counter app built using **React**.  
It demonstrates the use of the `useState` hook to manage state in a functional component.

---

## 🚀 Features

- ➕ Increment counter
- ➖ Decrement counter
- 🔁 Reset counter
- 🎨 Clean UI with inline styling

## Technologies Used

- React (with Vite)
- JavaScript
- HTML/CSS
